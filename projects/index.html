An AI-powered monitoring tool that uses computer vision to track student focus during study or online learning sessions. This project addresses the critical issue of distractions by employing machine learning algorithms to detect behaviors like yawning, phone use, or looking away from the screen.

Key Features

Real-time Detection

OpenCV & Mediapipe for live behavior analysis

Weekly Reports

Actionable insights to improve learning habits

Privacy-First

Smart Alerts

Instant feedback on attention lapses

Phase 2 Distraction logic integration
